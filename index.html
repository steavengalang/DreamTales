FAQ

Apakah DreamTales gratis?▾
Aplikasi berjalan lokal di browser. Untuk generate cerita kamu bisa pakai API key Groq saat testing.
Data saya disimpan di mana?▾
Data (tema, preset, library) disimpan di localStorage dan dipisahkan per pengguna (login).
Bagaimana cara mulai?▾
Klik tombol “Mulai DreamTales” atau buka halaman pembuat cerita. Kamu bisa langsung mencoba tanpa login sebagai Guest.
